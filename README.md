## ‚ö° Quick Start

### Prerequisites
- Node.js (v16+)
- MongoDB (for backend)

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/bike-shop-admin-dashboard.git
cd bike-shop-admin-dashboard
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash
npm run dev
```
